Class: ModelsStore

composables/models~ModelsStore()

Manages the model registry, standard benchmark inputs, audio sample catalogue, and badge helper utilities used by the template. State is initialised empty and populated by calling ModelsStore#init.

Constructor

new ModelsStore()

Creates a ModelsStore instance with empty reactive state. Call ModelsStore#init inside `onMounted` to load data.
