Class: BenchmarkStore

composables/benchmark~BenchmarkStore()

Manages the benchmark run lifecycle: form state, job submission, SSE streaming, and poll-based fallback when SSE is unavailable. Completion is signalled by setting BenchmarkStore#lastResult. `app.js` watches this ref to trigger cross-store effects (history refresh, tab switch) without creating a direct dependency between stores.
