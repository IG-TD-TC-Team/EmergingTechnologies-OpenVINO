Class: HistoryStore

composables/history~HistoryStore()

Manages the list of past benchmark runs and the currently displayed result. Also provides the JSON export action. The active result is written directly by `app.js` when a new benchmark completes, in addition to being loaded on demand via HistoryStore#loadResult.
