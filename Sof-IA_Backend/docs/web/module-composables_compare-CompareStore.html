Class: CompareStore

composables/compare~CompareStore()

Manages the selection of two past runs for side-by-side comparison and derives the comparison table rows automatically. The `compareRows` computed ref is recalculated whenever `compareResults` changes — no manual refresh needed.

(async) load() → {Promise.<void>}

Fetch both selected results in parallel and store them in `compareResults`. Requires both CompareStore#compareA and CompareStore#compareB to be set before calling. Errors are caught and logged — `compareResults` is not modified on failure.
